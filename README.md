# Portfólio de João Pedro

Este portfólio foi desenvolvido para apresentar meus projetos acadêmicos e profissionais, habilidades em engenharia de software, e formas de contato. O site possui um design moderno com foco em *Dark Mode*, uso de *Glassmorphism* e animações responsivas de rolagem (*Scroll Reveal*).

## Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web nativas (*vanilla*), garantindo alta performance, flexibilidade e facilidade de manutenção sem a necessidade de frameworks pesados:

- **HTML5**: Estruturação semântica e acessível.
- **CSS3**: Estilização baseada em variáveis globais para controle de temas, *Flexbox*, *CSS Grid*, e efeitos visuais avançados como `backdrop-filter`.
- **JavaScript (Vanilla)**: Lógica para animações baseadas em rolagem, comportamento suave de navegação (*Smooth Scrolling*) e integração com a *Clipboard API* para cópia de contato.

## Como executar localmente

Como o projeto consiste apenas em arquivos estáticos (Frontend), não há necessidade de configurar um ambiente de desenvolvimento complexo como Node.js ou Webpack.

1. Clone o repositório em sua máquina local:
   ```bash
   git clone https://github.com/johnaopedro/portfolio.git
   ```
2. Abra o arquivo `index.html` em seu navegador favorito ou utilize uma extensão de *Live Server* em sua IDE (como o VS Code) para ter atualizações em tempo real enquanto visualiza as modificações.

## Publicação (Deploy)

Este site foi arquitetado para ser hospedado rápida e gratuitamente utilizando o **GitHub Pages**. Por ser uma aplicação estática, qualquer atualização ou *commit* enviado para a branch `main` do repositório pode ser configurado para atualizar a página ao vivo instantaneamente.

Para acessar a versão final, visite: [johnaopedro.github.io](https://johnaopedro.github.io/portfolio/)
